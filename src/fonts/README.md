# Fonts

## Tamzen8x16.psf

The framebuffer console (`src/drivers/console.zig`) embeds this bitmap font.

- **Font:** Tamzen (8×16, regular), from
  [sunaku/tamzen-font](https://github.com/sunaku/tamzen-font), a refresh of the
  Tamsyn font by Scott Fial.
- **License:** see [`LICENSE.tamzen`](LICENSE.tamzen) — Tamsyn and Tamzen are
